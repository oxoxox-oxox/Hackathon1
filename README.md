我当前仓库需要实现一个VR辅助背单词的前后端网站项目

## 技术栈

## 前端要求

- 使用three.js和webxr实现VR环境中的单词展示和交互。

- 每个单词都从model文件夹里面读取对应名称的3D模型，并在VR环境中展示。

- 当用户用手柄点击单词时，显示单词的详细信息和相关的3D模型。

- 当用户用手柄再次点击3D模型的时候，展示和此单词相关的其他单词的3D模型，消除原本的3D模型。

- 留有“下一个”按钮，点击后展示下一个单词和相关的3D模型。

## 后端要求

- 使用Node.js搭建服务器，提供API接口供前端调用。

- 实现一个API接口，接受单词请求，调用model中的模型，并返回给前端。

- 返回单词的信息（为json文件）

## 其他要求

- 项目结构清晰，代码规范，易于维护和扩展。

- 要求同局域网下其他设备可以连接到本设备的前端服务器，进行VR体验。（用@vitejs/plugin-basic-ssl插件实现）
